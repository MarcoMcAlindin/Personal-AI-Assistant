# Cloud Gateway Backend Domain

**Agent Owner:** Mr. Green (Cloud Backend & API Engineer)
**Tech Stack:** Python + FastAPI

This directory houses the 24/7 cloud gateway deployed on Google Cloud Run. It is the sole bridge between the frontend clients and the external APIs or database.

Key responsibilities include defining the REST endpoints, proxying the Gmail API securely, parsing external RSS/XML feeds, and orchestrating Retrieval-Augmented Generation (RAG) using pgvector embeddings.
