# Mobile Frontend Domain

**Agent Owner:** Mr. Blue (Frontend & Mobile Architect)
**Tech Stack:** React Native (Expo)

This directory contains the mobile application for SuperCyan. It is optimized for one-handed use via a tab-based navigation system and strictly adheres to the OLED dark theme.

Critically, this app manages the foreground biometrics sync: retrieving data from `react-native-health-connect` on app open and pushing it to the cloud backend.
